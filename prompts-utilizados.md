**Prompts Utilizados**

Prompt 1 — Estrutura do eBook
Faça um texto para um ebook sobre lógica de programação para iniciantes. 

Regras:
> explique sempre de maneira simples;
> deixe o texto enxuto;
> sempre traga exemplos reais em código;
> crie um nome para o ebook.

Prompt 2 — Conteúdo Enxuto
Reescreva o conteúdo do ebook de forma mais enxuta, 
mas sem perder o aprendizado.

Foco: caber em poucas páginas do powerpoint, 
leitura fácil e direta.

Prompt 3 — Capa em Pixel Art (Tema RPG)
Gere um prompt para criar uma capa em pixel art estilo RPG,
para um ebook de lógica de programação.

Formato A4, sem texto, apenas a arte visual.

Prompt 4 — Rodapé em Pixel Art
Gere um prompt para criar uma imagem em pixel art 
com fundo branco e largura A4, 
tema programação, para usar como rodapé do ebook.
